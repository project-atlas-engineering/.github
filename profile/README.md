# Project Atlas

Project Atlas is a long-term engineering initiative focused on building a cohesive ecosystem of portable computing, networking, radio, and infrastructure systems.

Atlas is built with an emphasis on:

- thoughtful engineering
- reproducible systems
- maintainable software
- thorough documentation

## Repositories

- atlas-handbook — project philosophy, architecture, and engineering standards
- atlas-bootstrap — provisioning and configuration for Atlas Linux systems

Additional repositories will be added as the platform evolves.
